# 每日科技动态

WWDC26 开幕当天，Apple 发布了 Safari 27 beta，带来 58 项新特性与 525 项修复——这是近年 Safari 单次发布中修复量最大的一版。同一天，OpenAI 正式向 SEC 提交机密 S-1 文件，开启了可能是史上规模最大的科技 IPO 之路。前端生态在标准化与品质打磨上齐头并进，DevOps 侧则继续向更细粒度的安全与智能辅助演进。

## 今日概览

### 前端

**Safari 27 beta 是本次发布中最值得前端开发者关注的大版本。** 在 WWDC26 首日，WebKit 团队交出了近年质量打磨最深的一版 Safari：58 项新特性、525 项修复、4 项废弃。亮点包括可自定义样式的 `<select>` 元素——开发者期待已久的能力终于落地，无需再用 JavaScript 重建下拉控件即可匹配设计系统。CSS 侧新增 `:open` 伪类、`if()` 条件函数与 `stretch` 盒模型对齐关键字，`transform-aware anchor positioning` 补齐了锚点定位的重要缺口。WebAssembly JavaScript Promise Integration (JSPI) 让 Wasm 代码可以参与 JavaScript 的异步世界，Scroll Anchoring 则解决了内容异步加载时的视觉跳动问题。WebKit 团队在文末特意强调：这一版最大的工作不是「新东西」，而是让已有能力更正确、更一致地互操作。[来源](https://webkit.org/blog/17967/news-from-wwdc26-webkit-in-safari-27-beta/)

**Chrome 150 已进入 beta，计划 6 月 17 日正式发布。** 根据官方搜索摘要，Chrome 150 beta 带来了 `AccentColor` 与 `AccentColorText` 系统颜色，允许 Web 内容读取用户设备的系统强调色，让 PWA 等安装态应用获得接近原生应用的视觉一致性。CSS 方面新增 `image()` 函数（用任意颜色生成纯色图像）、`light-dark()` 对图像值的支持（可根据用户主题自动切换 `url()` / `image-set()`），以及 `background-clip: border-area`。此外，`polygon()` CSS 形状函数现在支持可选的圆角参数，CSS `zoom` 属性变为可动画插值。[来源](https://developer.chrome.com/blog/chrome-150-beta)

### DevOps

**GitHub Enterprise Cloud 的 IP 允许列表已覆盖 EMU 命名空间，正式进入 GA。** 企业托管用户（EMU）现在可以使用 GitHub 原生 IP 允许列表配置来保护用户命名空间级别的访问控制。对于需要满足严格合规要求的大型企业来说，这是一个将网络安全策略向下延伸到用户层的关键能力。[来源](https://github.blog/changelog/2026-06-08-ip-allow-list-coverage-for-emu-namespaces-in-general-availability)

**GitHub Copilot 获得更大的上下文窗口和可配置推理级别。** 6 月 4 日起，Copilot 支持百万 token 级上下文窗口，用户还可以按场景配置推理深度，在速度与深度之间自主取舍。这对于需要跨文件重构、大规模代码审查或复杂架构调整的场景尤其有用。同批更新还包括 Agent tasks REST API 进入公开预览（支持通过 API 启动和追踪 Copilot 云端 agent 任务），以及 Actions 失败后一键使用「Fix with Copilot」请求云端 agent 修复。[来源](https://github.blog/changelog/2026-06-04-larger-context-windows-and-configurable-reasoning-levels-for-github-copilot)

### AI

**OpenAI 正式向 SEC 提交机密 S-1 文件。** 6 月 8 日，OpenAI 官方确认已向美国证券交易委员会提交机密 S-1 注册声明，但未公布后续时间表。根据早前 Axios 与 CNBC 的报道（5 月 20 日），此次 IPO 估值可能突破 1 万亿美元，有望成为史上最大科技 IPO 之一。OpenAI 同时发布了「Built to benefit everyone」愿景文章，围绕可及性、安全性与共享繁荣阐述了其对 AGI 的公共承诺。[来源](https://openai.com/index/openai-submits-confidential-s-1)

**ChatGPT 推出「Dreaming」记忆系统，上下文保持能力大幅提升。** 6 月 4 日上线的 Dreaming 机制让 ChatGPT 能更好地记住用户偏好，在跨对话场景中保持上下文的新鲜度和相关性。这不是简单的「记住上次聊了什么」，而是对长期记忆机制的一次架构性升级，让模型能更主动地关联用户历史信息，减少重复说明成本。[来源](https://openai.com/index/chatgpt-memory-dreaming)

## 今日观察

前端领域正在进入一个「品质研磨期」。Safari 27 beta 的 525 项修复远超新特性数量，反映出浏览器厂商正把重心从堆砌新 API 转向让现有标准更稳定可靠。Chrome 150 的 AccentColor 系统颜色则指向另一个趋势：Web 应用正越来越深地嵌入操作系统生态，模糊浏览器与原生应用的边界。

DevOps 的精细化治理趋势明显——从 IP 级访问控制到可配置的 AI 推理深度，企业正获得更多「拧螺丝」的空间，而不再只是二选一的开关。

AI 行业的分水岭事件正在集中爆发。OpenAI 的 S-1 提交标志着 AI 头部公司从「技术叙事」正式转向「资本市场叙事」，而同一天发布的安全与公共愿景也透露出监管前的主动姿态。

## 简讯

- **前端**：WWDC26 开幕日发布 Safari 27 beta（58 新特性 / 525 修复），可自定义 `<select>`、CSS `:open` 伪类和 Wasm JSPI 成为本版亮点；Chrome 150 beta 新增 AccentColor 系统颜色与 CSS `image()` 函数。
- **DevOps**：GitHub EMU IP 允许列表正式 GA，Copilot 获得百万 token 上下文窗口与可配置推理级别。
- **AI**：OpenAI 向 SEC 提交机密 S-1 启动 IPO 进程；ChatGPT 上线 Dreaming 长期记忆系统。
